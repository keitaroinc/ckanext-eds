{{ _('What are Groups?') }}

{% trans %} Used to group data sets according position in the energy system. It may be within, for example, energy production, energy consumption, energy transport and market. A data set may belong to several groups. Some of the groups are considered as main groups, which form the data guide on the front page. All data sets are associated with at least one of the main groups, so that all data sets appear through the data guide. {% endtrans %}

{% trans %} The groups that are not displayed in the data guide are used to further filter the list of data sets. Finally all groups can also be accessed directly in the list of groups. {% endtrans %}
